The toggle switch component renders a SKY UX-themed switch for values that can either be "on" or "off." When users interact with a toggle switch, its value is driven through an ngModel attribute that you specify on the sky-toggle-switch element.

Sets the aria-label attribute of the toggle switch. Only use this value if the `sky-toggle-switch-label` component is not used. This property accepts string values. Indicates whether the toggle switch is selected. This property accepts boolean values. Indicates whether to disable the toggle switch. This property accepts boolean values. Specifies a tab index for the toggle switch. If not defined, the index is set to the position of the toggle switch on load. This property accepts number values. Fires when the checked state of a toggle switch changes. The event provides an argument with the SkyToggleSwitchChange type. References the checked state of the toggle switch.
